Management project system

=================================


SUPPORTED OPERATING SYSTEMS :

Management project system works on :

- Windows 8/10

REQUIREMENTS :

- WampServer
- Composer
- GitHub

BUILDING AND INSTALLING :

- Install WampServer

Download and install WampServer via this URL : http://www.wampserver.com/
- Install Composer

Download and install Composer via this URL: Composer
- Laravel installation

To install Laravel 5.3 visit this website : Laravel installation

You need also to enable those extensions :

OpenSSL PHP Extension
PDO PHP Extension
Mbstring PHP Extension
Tokenizer PHP Extension

This video will show you how to enable those extensions with WampServer: Youtube

- Download GitHub
Visit this website, download and install GitHub :GitHub

- Download our project
Clone our project in the www WampServer directory

Visit this website to see how to clone a project : Git Clone

The URL of our project is :
Demo Depot: Demo depot

Our project :
Then go in the www/Conduite_De_Projet_Demo_Equipe2/Src folder and tape this line in a terminal :
-composer install
- php artisan migrate
- php artisan db:seed
The client part is now available.

To see a result in the navigator :
- php artisan serve
go to Register

To test registration you have to download maildev and make some configurations. You can watch this video to install maildev Video

If you have any problems in facebook authentification :
-Download this file -Past the file in ...\wampp\php\ext\cacert.pem for windows and ...\xampp\php\ext\cacert.pem for mac and change extension to .pem
-Open php.ini file and copy this :

[cURL]
curl.cainfo="...\wampp\php\ext\cacert.pem"

you have to change the path

-Restart service of your server.
-Restart Laravel server to with php artisan config:cache then php artisan server
